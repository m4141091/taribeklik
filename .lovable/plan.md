

## קו מונפש בין אלמנטים בדף הבית

### מה ייבנה
קו מקווקו מעוגל ומונפש שמחבר בין 6 אלמנטים בדף הבית לפי הסדר הבא:
1. פירות (למטה)
2. ירקות (למטה)
3. עלים (למטה)
4. איקון אתר
5. איקון אריזה
6. איקון בית

הקו יהיה כהה, מקווקו, עם אנימציית "ציור" כשהמשתמש גולל לאזור.

### גישה טכנית
שימוש ב-SVG path ישירות ב-React (ללא ספריות חיצוניות) - יותר תואם לארכיטקטורה הקיימת מאשר LeaderLine שדורש DOM ישיר.

### קבצים שייווצרו/ישתנו

**קובץ חדש: `src/components/homepage/AnimatedPathLine.tsx`**
- קומפוננטת SVG שמציירת path מעוגל (cubic bezier curves) בין נקודות מוגדרות מראש
- הנקודות מחושבות כאחוזים מתוך גובה/רוחב הקונטיינר, בהתאם למיקומי האלמנטים בדאטאבייס
- מסלול הקו:
  - מתחת לפירות (81%, ~30%) --> יורד ועוקף שמאלה
  - מתחת לירקות (51%, ~30%) --> ממשיך שמאלה
  - מתחת לעלים (21%, ~30%) --> יורד
  - איקון אתר (19%, ~37%) --> עוקף ימינה
  - איקון אריזה (59%, ~38%) --> ממשיך ימינה ויורד
  - איקון בית (77%, ~45%)
- סגנון הקו: stroke-dasharray לקווקוו, צבע כהה (#251F20), עובי 3-4px
- אנימציה: stroke-dashoffset עם IntersectionObserver - הקו "מצטייר" כשנכנס לתצוגה
- נקודה עגולה (disc) בסוף הקו

**שינוי: `src/components/homepage/HomepageViewer.tsx`**
- ייבוא והוספת `AnimatedPathLine` כשכבה מעל האלמנטים
- הקומפוננטה תקבל את bgHeight לחישוב מיקומים נכון

### פרטי האנימציה
- הקו מוסתר בהתחלה (stroke-dashoffset שווה לאורך הקו)
- כשהמשתמש גולל לאזור, IntersectionObserver מפעיל אנימציית CSS שמורידה את ה-dashoffset ל-0
- משך האנימציה: כ-2 שניות עם easing חלק
- הקווקוו עצמו גם מונפש (dash animation רצה)

### חישוב הנתיב
הנתיב ישתמש ב-SVG cubic bezier curves כדי ליצור עקומות חלקות בין הנקודות, בדומה לצילום המסך מהפיגמה. הערכים המדויקים ייקבעו לפי מיקומי האלמנטים בדאטאבייס.
